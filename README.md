LANG v1.0: The Semantic Shell Launch
"Talk to your computer. It finally understands."
LANG is a high-performance Natural Language Operating Interface built in Rust. It replaces rigid terminal syntax with fluid, human-centric intent. This version is optimized specifically for Linux/WSL environments.
✨ What’s Working in v1.0?
📁 Structural Intelligence (Folders):
Full support for directory management. You can Create, Delete, Rename, and List folders using natural English (e.g., "Make a folder called Projects").
📄 Document Control (Files):
Create, Read, Delete, and Rename files instantly. The engine uses a local semantic mapper to ensure your hardware executes exactly what you intended.
📖 Instant View:
Use commands like "Show me what's inside [filename]" to peek at the contents of your documents directly in the terminal.
📝 Quick Append (Note Taking):
Add short lines of text or logs to existing files. Note: v1.0 is optimized for short-form data entry and system commands rather than long-form word processing.
🧠 Local & Private AI:
Powered by a local inference engine. Your commands never leave your machine—no internet required, 100% private.
🛠️ Current Platform Support
Linux / Ubuntu / WSL: Fully Supported (Native Binary).
Windows / Mac: Incoming in v2.0.
